# Portfólio do Bruno Alves Lopes

Projeto de portfólio pessoal desenvolvido com React, TypeScript, Vite e Tailwind CSS.

## O que este projeto possui

- Página inicial com apresentação profissional
- Seções de sobre, experiência, projetos e habilidades
- Assistente virtual local para responder perguntas sobre a trajetória do Bruno
- Layout responsivo e leve

## Como rodar

```bash
npm install
npm run dev
```

## Build de produção

```bash
npm run build
npm run preview
```
